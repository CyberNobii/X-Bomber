## üöÄ Installation

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/CyberNobii/X-bomber.git](https://github.com/CyberNobii/X-bomber.git)
    ```
2.  **Navigate to the Directory:**
    ```bash
    cd X-Bomber
    ```
3.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
4.  **Run the Bomber:**
    ```
    python3 bomber.py
    ```
